## 评委体验指南

### 本地运行 Demo

```bash
cd /home/shijialin/paperworking/workflow_new/productinsight-agent

# 终端 1：启动后端
conda activate py10
python -m uvicorn backend.app.main:app --reload --port 8005

# 终端 2：启动前端
streamlit run frontend/app.py --server.port 8505
```

访问 `http://localhost:8505` 体验完整流程。
