# Demo 录制指南 — ProductInsight Agent

> **推荐方式：自动化录屏脚本**（见下方「全程自动化录屏」章节）
> 全程录制 + 自动开关 OBS，无需手动操作
>
> 备用方式：OBS Studio 手动录屏（见「手动录屏」章节）
>
> 录制时长：完整 workflow 约 15-20 分钟，或用 Golden Demo 快速模式 3-5 分钟

---

## 一、全程自动化录屏（推荐）

### 1.1 功能概述

`scripts/record_demo.py` 实现全自动录屏流程：

1. 检查所有服务是否就绪（backend、Streamlit、DAG app）
2. 自动加载 Golden Demo 数据（预置数据，无需等待爬取）
3. 自动打开演示浏览器标签页（Streamlit、Report Viewer、DAG 可视化）
4. 通过 OBS WebSocket 自动开始/停止录制（无需手动按 F9/F10）
5. 实时打印 workflow 进度（当前节点、完成状态）
6. workflow 结束后自动停止录屏，保存文件

**特点**：一次性启动，人工零干预，录到完整 workflow。

### 1.2 前置准备

```bash
cd /home/shijialin/paperworking/workflow_new/productinsight-agent
conda activate py10

# 1. 安装依赖
pip install websocket-client requests

# 2. OBS Studio 配置（仅需做一次）
#    打开 OBS → Settings → Advanced → WebSocket server
#    ☑ Enable WebSocket server
#    ☑ Enable authentication（可选，设密码后用 --obs-password 传入）
#    Port: 4455

# 3. 确认服务已启动（如已在运行可跳过）
uvicorn backend.app.main:app --reload --port 8005 &
streamlit run frontend/app.py --server.port 8501 &
streamlit run frontend/views/report_viewer.py --server.port 8502 &
cd frontend/dag-app && npm run dev &
```

### 1.3 运行录屏

```bash
# 方式 A：OBS WebSocket（推荐，自动开关录制）
python scripts/record_demo.py --max-wait 20

# 方式 B：OBS 已设置密码
python scripts/record_demo.py --max-wait 20 --obs-password "your_password"

# 方式 C：ffmpeg 兜底（OBS 不可用时自动回退）
python scripts/record_demo.py --ffmpeg-fallback --max-wait 20

# 方式 D：测试模式（只检查服务，不断开录制）
python scripts/record_demo.py --dry-run
```

### 1.4 录屏进行中

脚本会：
- 实时打印当前节点名和状态
- 每 15 秒刷新一次进度
- workflow 结束后自动停止 OBS

```
  [00:15] status=running | current=build_task_brief | last_completed=-
  [00:30] status=running | current=plan_schema | last_completed=build_task_brief
  [01:45] status=running | current=collect_sources | last_completed=plan_schema
  ...
```

### 1.5 录制输出

视频文件保存至：
```
productinsight-agent/recordings/productinsight_demo_YYYYMMDD_HHMMSS.mp4
```

### 2.2 OBS Studio 手动设置

1. **窗口捕获**：选择 Chrome/Edge 浏览器窗口
2. **分辨率**：1920x1080（推荐）或 1280x720
3. **帧率**：30fps
4. **编码**：H.264（NVIDIA/AMD 硬件加速）
5. **输出**：MP4 文件
6. **快捷键**：
   - F9 开始/停止录制
   - F10 暂停
7. **WebSocket（可选）**：Settings → Advanced → WebSocket server → Enable（配合 `record_demo.py` 使用）
